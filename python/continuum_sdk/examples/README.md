# Continuum SDK Examples

## Quick Start (3 steps)

```python
from continuum import Agent

agent = Agent()
result = agent.run("hello")
```

## Examples

### hello_world.py
最简单的入门示例，展示3步启动。

### hello_agent.py
完整的Agent示例，包含会话管理和对话流程。

### session_example.py
Session类的详细使用示例。

### custom_tools.py
自定义工具的注册和使用。

### workflow.py
工作流编排示例。
