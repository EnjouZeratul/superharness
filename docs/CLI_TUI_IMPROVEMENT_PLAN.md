## 一、CLI 改进计划

### 1.1 默认行为改进

**当前行为**:
```bash
superharness              # 显示帮助信息
superharness tui          # 进入TUI
superharness run "task"   # 执行任务
```

**目标行为**:
```bash
superharness              # 直接进入TUI交互模式
superharness help         # 显示帮助信息
superharness run "task"   # 执行任务（保持不变）
superharness config ...   # 配置命令（保持不变）
```

### 1.2 实现方案

**修改文件**: `cli/src/cli/args.rs`

```rust
// 当前代码
#[command(subcommand)]
pub command: Commands,

// 改进代码
#[command(subcommand)]
pub command: Option<Commands>,

// 在 main.rs 中处理
match args.command {
    None => {
        // 无参数时启动 TUI
        Commands::Tui { session: None }
    }
    Some(cmd) => cmd,
}
```

### 1.3 新增 help 子命令

```rust
/// 显示详细帮助信息
Help {
    /// 显示特定命令的帮助
    #[arg(short, long)]
    command: Option<String>,
},
```
