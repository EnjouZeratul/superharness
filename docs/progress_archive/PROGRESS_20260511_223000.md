## 三、配置系统设计

### 三层配置方案

```
┌─────────────────────────────────────────────────────────────────┐
│  Layer 1: 环境变量 (零配置启动)                                  │
│  CONTINUUM_PROVIDER / API_KEY / BASE_URL / MODEL             │
├─────────────────────────────────────────────────────────────────┤
│  Layer 2: 配置文件 (持久化)                                      │
│  ~/.continuum/config.toml                                    │
│  支持 ${VAR_NAME} 环境变量引用                                   │
├─────────────────────────────────────────────────────────────────┤
│  Layer 3: CLI命令 (便捷操作)                                     │
│  config init / add-provider / use / show / list                 │
└─────────────────────────────────────────────────────────────────┘

优先级: 环境变量 > 配置文件 > 默认值
```

### CLI 命令示例

```bash
continuum config init                    # 初始化
continuum config add-provider custom \
  --key $KEY --url $URL --model glm-5      # 添加提供商
continuum config use custom              # 切换
continuum config show                    # 显示当前
continuum config list                    # 列出全部
```
