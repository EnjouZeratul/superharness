### CLI 使用示例

```bash
# 初始化配置
superharness config init

# 添加提供商
superharness config add-provider custom \
  --key $KEY --url https://api.example.com --model glm-5

# 切换提供商
superharness config use custom

# 查看配置
superharness config show
superharness config list
```
